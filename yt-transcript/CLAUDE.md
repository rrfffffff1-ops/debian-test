# yt-transcript

This is a Next.js app that fetches YouTube video transcripts and generates AI summaries using the Blackbox API.

## Commands
- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run linter

## Env
- `BLACKBOX_API_KEY` - API key for Blackbox AI
- `BLACKBOX_API_URL` - Blackbox API URL (default: https://api.blackbox.ai/v1)
