# yt-transcript

YouTube video transcript extractor and AI summarizer built with Next.js 16.

## Features
- Extract transcripts from any YouTube video
- View transcripts in segmented (time-stamped) or full-text mode
- AI-powered summarization via Blackbox API
- Copy transcript to clipboard
- Dark mode support

## Project Structure
- `yt-transcript/` - Next.js application
- `.github/workflows/` - CI/CD pipeline
